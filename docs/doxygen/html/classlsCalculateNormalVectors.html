Detailed Description

template<class T, int D>
class lsCalculateNormalVectors< T, D >

This algorithm is used to compute the normal vectors for all points defined in the level set. The result is saved in the lsDomain and can be retrieved with lsDomain.getNormalVectors(). Since neighbors in each cartesian direction are necessary for the calculation, the levelset width must be >=3.
