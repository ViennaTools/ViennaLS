\subsubsection*{template$<$class T, int D$>$\newline
class ls\+Calculate\+Normal\+Vectors$<$ T, D $>$}

This algorithm is used to find the normal vectors for all points with an and LS value less than 0.\+5 and only thos points. Since neighbors in each cartesion direction are necessary for the calculation, the levelset width must be $>$=3.
