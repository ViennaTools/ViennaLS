Vienna\+LS is a header-\/only C++ level set library developed for high performance topography simulations. The main design goals are simplicity and efficiency, tailored towards scientific simulations. Vienna\+LS can also be used for visualisation applications, although this is not the main design target.

I\+M\+P\+O\+R\+T\+A\+NT N\+O\+TE\+: Vienna\+LS is under heavy development and improved daily. If you do have suggestions or find bugs, please let us know!\hypertarget{md__home_klemenschits_Code_ViennaLS_README_autotoc_md1}{}\doxysection{Support}\label{md__home_klemenschits_Code_ViennaLS_README_autotoc_md1}
For help with getting started, have a look at the \href{https://github.com/ViennaTools/ViennaLS/tree/master/Examples}{\texttt{ Examples}}.

Bug reports and suggestions should be filed on Git\+Hub.\hypertarget{md__home_klemenschits_Code_ViennaLS_README_autotoc_md2}{}\doxysection{Contributing}\label{md__home_klemenschits_Code_ViennaLS_README_autotoc_md2}

Since this is a header only project, it does not require any installation. However, we recommend the following procedure.

Make sure you have \href{https://github.com/ViennaTools/viennahrle}{\texttt{ Vienna\+H\+R\+LE}} installed on your system and run\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{git clone github.com/ViennaTools/ViennaLS.git}
\DoxyCodeLine{cd ViennaLS}
\DoxyCodeLine{mkdir build \&\& cd build}
\DoxyCodeLine{cmake .. -\/DCMAKE\_INSTALL\_PREFIX=/path/to/your/custom/install/}
\DoxyCodeLine{make install}
\end{DoxyCode}


This will install the necessary headers and C\+Make files to the specified path. If D\+C\+M\+A\+K\+E\+\_\+\+I\+N\+S\+T\+A\+L\+L\+\_\+\+P\+R\+E\+F\+IX is not specified, it will be installed to the standard path for your system, usually /usr/local/ .

In order to save build time during developement, shared libraries are built by default. This can be disabled by setting the C\+Make option V\+I\+E\+N\+N\+A\+L\+S\+\_\+\+B\+U\+I\+L\+D\+\_\+\+S\+H\+A\+R\+E\+D\+\_\+\+L\+I\+BS=O\+FF .\hypertarget{md__home_klemenschits_Code_ViennaLS_README_autotoc_md7}{}\doxysubsection{Building examples}\label{md__home_klemenschits_Code_ViennaLS_README_autotoc_md7}
The examples can be built using C\+Make\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{mkdir build \&\& cd build}
\DoxyCodeLine{cmake .. -\/DVIENNALS\_BUILD\_EXAMPLES=ON}
\DoxyCodeLine{make}
\end{DoxyCode}
